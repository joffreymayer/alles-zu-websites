Chapter 1 Set-Up

1.1 Backend: Git & Github

  1. Create a Github-Account.
  2. We also need to download Git, a Version-Control System, because you can track all your changes and it is easier to collaborate on Code-Projects.
  3. Next, we need to link our Github-Account to Git. To do this, use the Terminal and type in the following two commands:

3.1) git config --global user.name "YOUR GIBHUB USERNAME HERE" 3.2) git config --global user.email "your_github_email@example.com"

  1. Create a Public & Private SSH-Key in order to not always type in your password. Follow the instruction of this Youtube-Tutorial (ab 2:40-4:03).
  2. In order to be able to use the SSH-Key, you need to follow my instruction in the Video on my local computer, situated on the following path: /Users/jomaye/Nextcloud/Dokumente/Life/Video Tutorials/Github/ssh-key-zum-laufen-bringen.mov. If the Git code that I provided in the video does not work, then I recommend you to also try this code out: git remote set-url --add --push origin git@github.com:yourGithubUsername/yourRepo.git
